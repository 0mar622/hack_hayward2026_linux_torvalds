# AI Professional Conversation Assistant

## What it does
Analyzes professional conversations (networking, interview, pitch) and extracts structured insights like summaries, opportunities, and follow-ups.

## Features
- Networking insights (name, opportunity, follow-up)
- Interview feedback (clarity, confidence)
- Pitch feedback (investor feedback, action items)

## Tech Stack
- Frontend: React (Vite)
- Backend: FastAPI
- AI Logic: Python (rule-based NLP)

## How to run

### Backend
cd backend
python3 -m uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm run dev
